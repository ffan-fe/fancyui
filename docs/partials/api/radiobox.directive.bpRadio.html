bpRadio
directive in module radiobox

Description

包装了Input[type="radio"] 包含的状态有

  • enable
  • disabled
  • checked

Usage

as element:
<bp-radio
       disabled
       value="{String}"
       model="{ANY}"
       complex="{complex}">
</bp-radio>

Parameters

ParamTypeDetails
disabledBoolean
  • binding symbol is <, 禁用状态,默认 false
valueString
  • binding symbol is @, 选中后的值,默认 false
modelANY
  • binding symbol is =?, 绑定在radioBox上的model
complexcomplex
  • binding symbol is =?complex, 因为value只能提供String, 复杂结构数据类型使用, complex指令

Example

<bp-radiobox model="vm.radio" value="1"></bp-radiobox>
<bp-radiobox model="vm.radio" value="2"></bp-radiobox>
