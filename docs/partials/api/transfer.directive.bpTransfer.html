bpTransfer
directive in module transfer

Description

  • 双栏穿梭选择框
  • 用直观的方式在两栏中移动元素，完成选择行为

Usage

as element:
<bp-transfer
       source-data="{Array}"
       target-data="{Array}"
       titles="{Array}"
       placeholders="{Array}"
       operations="{Array}"
       show-left-search="{Boolean}"
       show-right-search="{Boolean}"
       add-btn="{Boolean}"
       add-all-btn="{Boolean}"
       del-btn="{Boolean}"
       del-all-btn="{Boolean}"
       server-search="{Function}">
</bp-transfer>

Parameters

ParamTypeDetails
sourceDataArray
  • binding symbol is =?, 数据源
targetDataArray
  • binding symbol is =?, 右侧选中目标值
titlesArray
  • binding symbol is <, 左右侧标题,默认为请选择
placeholdersArray
  • binding symbol is <, 左右placeholders值 默认为请输入检索条件
operationsArray
  • binding symbol is <, 操作按钮的文案 默认依次为 添加 添加全部 删除 全部删除
showLeftSearchBoolean
  • binding symbol is <, 是否启用左侧检索框 默认为 true 启用
showRightSearchBoolean
  • binding symbol is <, 是否启用右侧检索框 默认为 true 启用
addBtnBoolean
  • binding symbol is <, 是否展示增加按钮 默认为 true 展示
addAllBtnBoolean
  • binding symbol is <, 是否展示全部增加按钮 默认为 true 展示
delBtnBoolean
  • binding symbol is <, 是否展示删除按钮 默认为 true 展示
delAllBtnBoolean
  • binding symbol is <, 是否展示全部删除按钮 默认为 true 展示
serverSearchFunction
  • binding symbol is &?, 是否启用服务端检索

Example

<bp-transfer
source-data="vm.dataSource"
titles="vm.titles"
placeholders="vm.placeholders"
operations="vm.operations"
showLeft-search="vm.showLeftSearch"
showRight-search="vm.showRightSearch"
add-btn="vm.addBtn"
add-all-btn="vm.addAllBtn"
del-btn="vm.delBtn"
del-all-btn="vm.delAllBtn"
></bp-transfer>
